// THE EQUATIONS THAT POWER THE INTELLIGENCE

Not just a user of libraries. A student of the mathematics that makes them work — calculus, linear algebra, probability. Teaching math makes the code cleaner.

Euler's Identity
e^(iπ) + 1 = 0
The most beautiful equation in mathematics. The foundation of signal processing and Fourier transforms used in audio and image AI.
Gradient Descent
θ := θ − α·∇J(θ)
The engine of every neural network. Each parameter update drives the model toward minimum loss — the core of TensorFlow training loops.
Backpropagation
∂L/∂w = ∂L/∂a · ∂a/∂z · ∂z/∂w
The chain rule applied recursively. How neural networks learn by propagating errors backward through every layer.
GAN Objective
min_G max_D V(D,G)
The adversarial game between Generator and Discriminator. The mathematical heart of the Ladybird data augmentation pipeline.
Cross Entropy Loss
L = −Σ y·log(ŷ)
Measures divergence between predicted and true distributions. The loss function optimized during species classification training.
Softmax
σ(z)_i = e^(z_i) / Σe^(z_j)
Converts raw logits into a probability distribution across species classes. The final layer of every multi-class classifier.
